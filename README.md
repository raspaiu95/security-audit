# Security Audit Automation

## 1. 개요 (Overview)

이 프로젝트는 Linux 서버 환경(CentOS, Ubuntu, Rocky Linux)을 대상으로  
보안 취약점 진단을 자동 수행하고, 진단 결과를 표준화된 형태로 정리하기 위해 개발한
보안 점검 자동화 도구입니다.

운영 환경에서 반복적으로 수행되던 수동 점검 작업을 자동화하여
진단 소요 시간을 단축하고, 결과 정합성과 재사용성을 높이는 것을 목표로 합니다.

---

## 2. 개발 배경 (Background)

다수의 서버를 운영하는 환경에서 다음과 같은 문제를 경험했습니다.

- OS별로 상이한 점검 항목과 스크립트 관리 부담
- 진단 결과가 텍스트 파일 형태로 산출되어 후처리(취합/요약)가 어려움
- 동일 항목이 여러 번 등장하는 결과 파일로 인한 분석 오류
- 인프라 환경 변화에 따라 스크립트를 지속적으로 수정해야 하는 문제

이러한 문제를 해결하기 위해  
**OS별 점검 자동화 + 결과 파싱/정규화** 구조로 도구를 설계했습니다.

---

## 3. 주요 기능 (Features)

- Linux OS별 보안 취약점 자동 점검
  - CentOS
  - Ubuntu
  - Rocky Linux
- 점검 결과 텍스트 파일 자동 생성
- 진단 결과 파싱 및 표준 포맷 변환
- 중복 점검 항목 병합 및 위험도 기준 정리
- 파일명에서 호스트 정보(IP/Hostname) 자동 추출 및 정규화

---
